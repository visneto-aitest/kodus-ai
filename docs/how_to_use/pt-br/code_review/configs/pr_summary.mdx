---
title: "Sumarização de PR"
---

O recurso de Resumo de PR de Kody permite automatizar a criação de descrições de pull request concisas e acionáveis.
Este guia de configuração explica como adaptar o recurso ao fluxo de trabalho e preferências da sua equipe.

## Habilitando Resumos Automáticos

Habilite ou desabilite a geração automática de resumos de pull request:

Alterne a opção Habilitar Geração Automática de Resumo.

Quando habilitado, Kody gerará resumos para cada pull request, garantindo consistência e economizando tempo.

## Comportamento para Descrições Existentes

Controle como Kody interage com descrições de pull request existentes. Você pode escolher entre três modos:

### Substituir (Padrão):

Sobrescreve qualquer descrição existente com um resumo recém-gerado do zero.

### Concatenar:

Anexa o resumo gerado ao final da descrição existente.

### Complementar:

Combina os insights gerados por Kody com a descrição atual, destacando apenas o que é novo ou relevante.

## Instruções Personalizadas

Ajuste como Kody gera resumos fornecendo diretrizes específicas:

Adicione instruções diretamente no campo de texto Instruções Personalizadas.

**Exemplos:**

* "Foque nas alterações de segurança e nos impactos de performance."
* "Resuma de forma concisa, destacando as APIs públicas modificadas."
