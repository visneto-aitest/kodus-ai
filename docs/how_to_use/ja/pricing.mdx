組織にとって最良の決定を下すために必要な、価格に関するすべての情報をご紹介します。最新のプラン詳細とライブ価格については [kodus.io/pricing](https://kodus.io/pricing) をご覧ください。

## プランの概要

- **Community** — 無料。BYOK。セルフホストまたはクラウド。
- **Teams** — $10/開発者/月 + トークンコスト（BYOK）。クラウドのみ。**14日間の無料トライアル、クレジットカード不要。** 年間請求で20%オフ。
- **Enterprise** — カスタム価格。セルフホストまたはクラウド。**BYOK** または **Kodus AI トークン APIキー** を選択。

すべてのプランには無制限のユーザー、無制限のプルリクエスト、最小シート数なしが含まれます。

## Community vs. Teams vs. Enterprise

Kodus は3つのバージョンで提供されています:

- **Community:** 無料で使用・変更可能。セルフホストとクラウドデプロイメントの両方で利用可能。

  - 独自の APIキーを持ち込む（BYOK）
  - 無制限のコードレビューと無制限のユーザー
  - Quality Radar の無制限のイシュー
  - Kody の学習とメモリ
  - Kodus MCP
  - 最大10件の Kody Rules
  - 最大3件のプラグイン/MCP（デフォルトの Kodus MCP を含む）
  - ❌ Cockpit なし（Teams および Enterprise のみ）
  - Discord によるコミュニティサポート

- **Teams:** クラウド専用の有料バージョン。$10/開発者/月 + トークンコスト（BYOK）。

  - Community のすべての機能、さらに:
  - 無制限の Kody Rules（10ルール制限なし）
  - 無制限のプラグイン/MCP（3プラグイン制限なし）
  - **Kody エージェントの優先キュー**
  - **Kodus Cockpit**（エンジニアリングメトリクス）
  - メール + Discord サポート
  - 自動更新とメンテナンス
  - **14日間の無料トライアル** — クレジットカード不要
  - 年間請求で**20%割引**

- **Enterprise:** エンタープライズ用のカスタマイズバージョン。セルフホストまたはクラウドで利用可能。

  - Teams のすべての機能、さらに:
  - SSO / SAML
  - **RBAC、監査ログ、アナリティクス**
  - **SOC 2 コンプライアンス対応中**
  - トークンモデルの選択: **BYOK** または **Kodus AI トークン APIキー**（Kodus が請求するマネージドトークン）
  - 専用インスタンス
  - カスタム SLA
  - 月最大**5時間**の専用サポート
  - プライベート Discord チャンネル
  - 高度なセキュリティ機能
  - トレーニングとオンボーディングアシスタンス
  - カスタマイズオプション

## 価格の仕組み

価格はチーム内でアクティブにプルリクエストを作成する開発者ごとです。チームの進化に合わせて請求が価値と一致します:

- **アクティブな開発者ごとに課金:** PR を貢献する開発者のみがシートにカウントされます。
- **いつでも調整可能:** アカウント設定からアクティブな開発者の数を変更できます。最小値なし、シート数の年間ロックインなし。

## BYOK はすべてのプランでデフォルト

Kodus はデフォルトで LLM トークンを販売**しません**。Community、Teams、Enterprise はすべて **Bring Your Own Key**（BYOK）で動作します: 独自のプロバイダーアカウント（OpenAI、Anthropic、Gemini、または OpenAI 互換プロバイダー）を接続し、トークンはそのプロバイダーのダッシュボードで直接請求されます。Kodus はトークンに上乗せ料金を請求しません。

プラン間の価格差は、トークンモデル自体ではなく、**トークン費用に加えて Kodus が請求する金額**です:

| プラン | Kodus への支払い | トークンの請求方法 |
| ---- | ------------------ | --------------------- |
| **Community** | $0 | LLM プロバイダーに直接支払い（BYOK） |
| **Teams** | $10/アクティブ開発者/月 | LLM プロバイダーに直接支払い（BYOK） |
| **Enterprise** | カスタム（契約） | BYOK、**または** Kodus が **Kodus AI トークン APIキー** でトークンを管理して単一の請求書を受け取る |

サポートされているプロバイダーとセットアップフローについては [BYOK 概要](/how_to_use/ja/byok) を参照してください。以下はすべて **Kodus 側の料金**についてです。

## Teams 価格

### アクティブな開発者1人あたり月$10

Teams サブスクリプションは、LLM プロバイダーへのトークン費用に加えて、**アクティブな開発者1人あたり月$10**です。トークンへの Kodus の上乗せはありません。

$10/開発者/月でカバーされるもの:

- **クラウドホスティング** — 自動更新とメンテナンスによる Kodus の完全管理。
- **Kody エージェントの優先キュー** — Teams のトラフィックは無料ティアのトラフィックより先にレビューされます。
- **Kodus Cockpit** — エンジニアリングメトリクスとレポーティング。
- **無制限の Kody Rules とプラグイン/MCP**（10ルール/3プラグインの上限なし）。
- **メール + Discord サポート**。

トークンコストでカバーされるもの:

- レビュー中に Kodus が選択した LLM プロバイダーに行うすべての API コール。
- プロバイダーが直接請求; いつでもプロバイダーを切り替えられます。
- モデルを選択 — コスト効率の良いもの（例: Gemini Flash、GPT-5-mini）またはフロンティア（Claude 4.5 Sonnet、GPT-5）。

### 請求サイクル

- **月次:** $10/開発者/月、月ごとに請求。
- **年次:** 年1回請求の場合20%オフ。
- **無料トライアル:** Teams で14日間、クレジットカード不要。
- **いつでもキャンセル:** 変更は現在の請求サイクルの終了時に有効になります。

## Enterprise 価格

Enterprise はカスタム価格で契約を通じて締結されます。運用に合ったトークンモデルを選択できます:

- **BYOK** — Community および Teams と同じく、独自の LLM キーを持ち込みます。Kodus はシートごとの料金のみを請求します。
- **Kodus AI トークン APIキー** — Kodus がマネージド APIキーを提供し、シート料金とともにトークンを請求します。単一のベンダー、単一の請求書を望み、LLM プロバイダーとの直接の関係を望まないチームに最適。

Enterprise 専用機能には SSO / SAML、監査ログとアナリティクスを含む RBAC、専用インスタンスオプション、カスタム SLA、月最大**5時間**の専用サポート、プライベート Discord チャンネルが含まれます。SOC 2 コンプライアンスは対応中です。

組織に合わせた見積もりは [営業に連絡](https://kodus.io/pricing) してください。

## 無制限のプルリクエスト

チームは好きなだけプルリクエストを提出できます — どのプランにも PR 制限はありません。唯一のハード制限は**非常に大きなプルリクエスト**です: **200以上の変更ファイル**を含む PR はレビューしません。Git プロバイダー側のレート制限も適用される場合があります。

## Kody パイロットプログラム

業界を前進させるチームと、その他の方法ではアフォードできないチームのために Kodus を補助します。**Kody パイロットプログラム**は以下に対して**30〜100%の割引**を提供します:

- **アーリーステージのスタートアップ**
- アクティブな開発を持つ**オープンソースプロジェクト**
- **経済的に不利な地域のチーム**
- **非営利団体**

### 申請方法

1. [discord.gg/TFZBRk9fT6](https://discord.gg/TFZBRk9fT6) の Discord コミュニティに参加。
2. 組織名、ウェブサイト、業務の簡単な説明を共有。
3. 資格の証明を提供:
   - スタートアップ: ステージと資金調達ラウンド（ある場合）。
   - オープンソース: 公開リポジトリへのリンク。
   - 不利な地域: 国とコンテキスト。
   - 非営利: 非営利ステータスの証明。
4. 3営業日以内に申請を審査します。

## 支払いと請求

- **Stripe による処理:** 支払いは世界的に信頼されるプラットフォームである Stripe を使用して安全に処理されます。
- **いつでもキャンセル:** プランはいつでもキャンセルできます; 変更は請求サイクル（月次または年次）の終了時に有効になります。
- **請求書と Brazilian Nota Fiscal:** ブラジルのお客様 🇧🇷 には、すべての取引に対して請求書と Nota Fiscal（税務領収書）を発行します。
