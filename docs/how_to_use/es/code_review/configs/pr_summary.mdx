---
title: "Resumen de PR"
---

La función de Resumen de PR de Kody te permite automatizar la creación de descripciones de pull request concisas y accionables.
Esta guía de configuración explica cómo adaptar la función al flujo de trabajo y las preferencias de tu equipo.

## Habilitar Resúmenes Automáticos

Habilita o deshabilita la generación automática de resúmenes de pull request:

Activa la opción Habilitar Generación Automática de Resúmenes.

Cuando está habilitado, Kody generará resúmenes para cada pull request, asegurando consistencia y ahorrando tiempo.

## Comportamiento con Descripciones Existentes

Controla cómo Kody interactúa con las descripciones de pull request existentes. Puedes elegir entre tres modos:

### Reemplazar (Predeterminado):

Sobreescribe cualquier descripción existente con un resumen recién generado desde cero.

### Concatenar:

Agrega el resumen generado al final de la descripción existente.

### Complementar:

Combina los insights generados por Kody con la descripción actual, destacando solo lo que es nuevo o relevante.

## Instrucciones Personalizadas

Ajusta cómo Kody genera los resúmenes proporcionando pautas específicas:

Agrega instrucciones directamente en el campo de texto Instrucciones Personalizadas.

**Ejemplos:**

* "Enfócate en los cambios de seguridad y los impactos en el rendimiento."
* "Resume de forma concisa, destacando las APIs públicas modificadas."
