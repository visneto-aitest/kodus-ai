---
title: "Atualizando"
description: "Atualize sua instalação do Kodus."
icon: "arrow-up"
---

Para atualizar sua instalação do Kodus para a versão mais recente:

```bash
# Navegue até o diretório de instalação do Kodus
cd kodus-installer

# Obtenha as últimas alterações do repositório
git pull

# Se estiver fazendo upgrade da versão 1.x, atualize seu .env com as novas variáveis da versão 2.0
# (RabbitMQ, MCP Manager e webhooks).

# Re-execute o instalador para baixar imagens, iniciar serviços e executar migrações
./scripts/install.sh

# Opcional: executar diagnósticos
./scripts/doctor.sh
```
