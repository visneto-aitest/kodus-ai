## Como o Fireworks AI funciona

O Fireworks AI é a plataforma de inferência mais rápida para IA generativa, projetada para criar e executar aplicações de IA em segundos. A plataforma fornece acesso serverless a modelos open-source populares como DeepSeek, Llama, Qwen e Mistral com velocidade otimizada, alto throughput e latência mínima. Desenvolvido para desenvolvedores que precisam de uma infraestrutura de IA confiável e extremamente rápida sem a complexidade de gerenciar GPUs.

## Criando uma Chave de API

<Warning>Uma conta Fireworks AI é necessária para criar uma Chave de API.</Warning>

Acesse diretamente o [Console Fireworks AI](https://app.fireworks.ai) para criar uma nova Chave de API.

Ou siga estes passos:

1. Visite [app.fireworks.ai](https://app.fireworks.ai) e crie uma conta ou faça login
2. Após fazer login, navegue até a página de **Chaves de API** nas configurações da sua conta
3. Clique no botão **"Criar Chave de API"**
4. Dê um nome descritivo à sua chave (ex.: 'Kodus' ou qualquer nome que preferir)
5. Clique em **"Criar"** para gerar a chave
6. Copie a chave de API imediatamente e guarde-a em um lugar seguro - você não poderá vê-la novamente

<Info>
  Novas contas recebem $1 em créditos gratuitos para começar com seus projetos.
</Info>

### Configurar o Fireworks AI no Arquivo de Ambiente

Edite o seu arquivo `.env` e configure as definições principais. Para a **Integração de LLM**, use o Fireworks AI no Modo Fixo:

```env
# Configurações Principais do Sistema (atualize com seus domínios)
WEB_HOSTNAME_API="kodus-api.yourdomain.com"    
WEB_PORT_API=443                               
NEXTAUTH_URL="https://kodus-web.yourdomain.com"

# Chaves de Segurança (gere com os comandos openssl acima)
WEB_NEXTAUTH_SECRET="your-generated-secret"
WEB_JWT_SECRET_KEY="your-generated-secret"
API_CRYPTO_KEY="your-generated-hex-key"
API_JWT_SECRET="your-generated-secret"
API_JWT_REFRESHSECRET="your-generated-secret"

# Configuração do Banco de Dados
API_PG_DB_PASSWORD="your-secure-db-password"
API_MG_DB_PASSWORD="your-secure-db-password"

# Configuração Fireworks AI (Modo Fixo) 
API_LLM_PROVIDER_MODEL="accounts/fireworks/models/llama4-maverick-instruct-basic"  # Escolha seu modelo preferido
API_OPENAI_FORCE_BASE_URL="https://api.fireworks.ai/inference/v1"                  # URL da API Fireworks AI  
API_OPEN_AI_API_KEY="your-fireworks-api-key"                                       # Sua Chave de API Fireworks AI

# Webhooks do Provedor Git (escolha seu provedor)
API_GITHUB_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/github/webhook"
# ou API_GITLAB_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/gitlab/webhook"
# ou GLOBAL_BITBUCKET_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/bitbucket/webhook"
```

<Note>
  As URLs de webhook devem alcançar o serviço de Webhooks (porta 3332). Use um domínio dedicado para webhooks ou roteie `/.../webhook` para a porta 3332 no seu proxy reverso.
</Note>

<Info>
  **O Modo Fixo é ideal para o Fireworks AI** porque ele fornece APIs compatíveis com OpenAI com velocidades de inferência extremamente rápidas e acesso a modelos open-source de ponta sem tempo de configuração.
</Info>

### Configurar Proxy Reverso (Para Produção)

Para webhooks e acesso externo, configure o Nginx:

```nginx
# Aplicação Web (porta 3000)
server {
    listen 80;
    server_name kodus-web.yourdomain.com;
    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}

# API (porta 3001)  
server {
    listen 80;
    server_name kodus-api.yourdomain.com;
    location ~ ^/(github|gitlab|bitbucket|azure-repos)/webhook {
        proxy_pass http://localhost:3332;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location / {
        proxy_pass http://localhost:3001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### Verificar a Integração com o Fireworks AI

Além da verificação básica de instalação, confirme que o Fireworks AI está funcionando:

```bash
# Verificar especificamente a conexão com a API Fireworks AI
docker-compose logs api worker | grep -i fireworks
```

<Tip>
  Para informações detalhadas sobre configuração de SSL, monitoramento e configurações avançadas, consulte nosso [guia completo de implantação](https://docs.kodus.io/docs/how_to_deploy/en/deploy_kodus/generic_vm).
</Tip>

### Solução de problemas

<AccordionGroup>
  <Accordion title="Problemas com Chave de API">
    - Verifique se sua chave de API está correta e ativa no [Console Fireworks AI](https://app.fireworks.ai)
    - Verifique se você tem créditos suficientes na sua conta Fireworks AI
    - Certifique-se de que não há espaços extras no seu arquivo `.env`
    - Novas contas recebem $1 em créditos gratuitos para começar
  </Accordion>
  
  <Accordion title="Modelo Não Encontrado">
    - Verifique se o nome do modelo está escrito corretamente na sua configuração
    - Verifique se o modelo está disponível na biblioteca de modelos atual do Fireworks AI
    - Tente com um modelo diferente da nossa lista recomendada
    - Consulte a [documentação de modelos do Fireworks AI](https://docs.fireworks.ai/getting-started/recommended-open-models)
  </Accordion>
  
  <Accordion title="Erros de Conexão">
    - Verifique se o seu servidor tem acesso à internet para alcançar `api.fireworks.ai`
    - Verifique se há restrições de firewall
    - Revise os logs da API/worker para mensagens de erro detalhadas
    - Certifique-se de que está usando o endpoint de API correto
  </Accordion>
  
  <Accordion title="Problemas de Desempenho">
    - O Fireworks AI fornece velocidades líderes do setor com latência mínima
    - Verifique sua conectividade de rede para desempenho ideal
    - Considere usar implantações dedicadas para cargas de trabalho empresariais
    - Monitore seus padrões de uso para otimizar as chamadas de API
  </Accordion>
  
  <Accordion title="Limitação de Taxa">
    - O Fireworks AI fornece limites de taxa altos em infraestrutura serverless
    - Verifique seu uso atual no painel do Fireworks AI
    - Considere fazer upgrade para implantações dedicadas para maior throughput
    - Entre em contato com o suporte do Fireworks AI para ajustes de limite de taxa empresarial
  </Accordion>
</AccordionGroup>
