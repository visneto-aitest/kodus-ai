## Cómo funciona Fireworks AI

Fireworks AI es la plataforma de inferencia más rápida para IA generativa, diseñada para construir y ejecutar aplicaciones de IA en segundos. La plataforma proporciona acceso sin servidor a modelos de código abierto populares como DeepSeek, Llama, Qwen y Mistral con velocidad optimizada, alto rendimiento y latencia mínima. Construido para desarrolladores que necesitan infraestructura de IA confiable y ultrarrápida sin la complejidad de la gestión de GPUs.

## Crear Clave API

<Warning>Se requiere una cuenta de Fireworks AI para crear una clave API.</Warning>

Ve directamente a la [Consola de Fireworks AI](https://app.fireworks.ai) para crear una nueva clave API.

O sigue estos pasos:

1. Visita [app.fireworks.ai](https://app.fireworks.ai) y crea una cuenta o inicia sesión
2. Una vez dentro, navega a la página de **Claves API** en la configuración de tu cuenta
3. Haz clic en el botón **"Crear Clave API"**
4. Dale a tu clave un nombre descriptivo (por ejemplo, 'Kodus' o cualquier nombre que prefieras)
5. Haz clic en **"Crear"** para generar la clave
6. Copia la clave API de inmediato y guárdala en un lugar seguro — no podrás verla nuevamente

<Info>
  Las cuentas nuevas vienen con $1 en créditos gratuitos para comenzar con tus proyectos.
</Info>

### Configurar Fireworks AI en el Archivo de Entorno

Edita tu archivo `.env` y configura los ajustes principales. Para la **Integración de LLM**, usa Fireworks AI en Modo Fijo:

```env
# Configuración principal del sistema (actualiza con tus dominios)
WEB_HOSTNAME_API="kodus-api.yourdomain.com"    
WEB_PORT_API=443                               
NEXTAUTH_URL="https://kodus-web.yourdomain.com"

# Claves de seguridad (genera con los comandos openssl anteriores)
WEB_NEXTAUTH_SECRET="your-generated-secret"
WEB_JWT_SECRET_KEY="your-generated-secret"
API_CRYPTO_KEY="your-generated-hex-key"
API_JWT_SECRET="your-generated-secret"
API_JWT_REFRESHSECRET="your-generated-secret"

# Configuración de la base de datos
API_PG_DB_PASSWORD="your-secure-db-password"
API_MG_DB_PASSWORD="your-secure-db-password"

# Configuración de Fireworks AI (Modo Fijo) 
API_LLM_PROVIDER_MODEL="accounts/fireworks/models/llama4-maverick-instruct-basic"  # Elige tu modelo preferido
API_OPENAI_FORCE_BASE_URL="https://api.fireworks.ai/inference/v1"                  # URL de la API de Fireworks AI  
API_OPEN_AI_API_KEY="your-fireworks-api-key"                                       # Tu clave API de Fireworks AI

# Webhooks del proveedor de Git (elige tu proveedor)
API_GITHUB_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/github/webhook"
# o API_GITLAB_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/gitlab/webhook"
# o GLOBAL_BITBUCKET_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/bitbucket/webhook"
```

<Note>
  Las URLs de webhook deben alcanzar el servicio de Webhooks (puerto 3332). Usa un dominio dedicado para webhooks o enruta `/.../webhook` al puerto 3332 en tu proxy inverso.
</Note>

<Info>
  **El Modo Fijo es ideal para Fireworks AI** porque proporciona APIs compatibles con OpenAI con velocidades de inferencia ultrarrápidas y acceso a modelos de código abierto de vanguardia sin tiempo de configuración.
</Info>

### Configurar Proxy Inverso (Para Producción)

Para webhooks y acceso externo, configura Nginx:

```nginx
# Aplicación web (puerto 3000)
server {
    listen 80;
    server_name kodus-web.yourdomain.com;
    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}

# API (puerto 3001)  
server {
    listen 80;
    server_name kodus-api.yourdomain.com;
    location ~ ^/(github|gitlab|bitbucket|azure-repos)/webhook {
        proxy_pass http://localhost:3332;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location / {
        proxy_pass http://localhost:3001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### Verificar la Integración con Fireworks AI

Además de la verificación básica de instalación, confirma que Fireworks AI está funcionando:

```bash
# Verificar específicamente la conexión con la API de Fireworks AI
docker-compose logs api worker | grep -i fireworks
```

  Para información detallada sobre configuración de SSL, monitoreo y configuraciones avanzadas, consulta nuestra [guía de despliegue completa](https://docs.kodus.io/docs/how_to_deploy/en/deploy_kodus/generic_vm).

### Solución de problemas

<AccordionGroup>
  <Accordion title="Problemas con la clave API">
    - Verifica que tu clave API sea correcta y esté activa en la [Consola de Fireworks AI](https://app.fireworks.ai)
    - Comprueba si tienes créditos suficientes en tu cuenta de Fireworks AI
    - Asegúrate de que no haya espacios extra en tu archivo `.env`
    - Las cuentas nuevas reciben $1 en créditos gratuitos para comenzar
  </Accordion>
  
  <Accordion title="Modelo no encontrado">
    - Verifica que el nombre del modelo esté correctamente escrito en tu configuración
    - Comprueba que el modelo esté disponible en la biblioteca de modelos actual de Fireworks AI
    - Prueba con un modelo diferente de nuestra lista de recomendados
    - Consulta la [documentación de modelos de Fireworks AI](https://docs.fireworks.ai/getting-started/recommended-open-models)
  </Accordion>
  
  <Accordion title="Errores de conexión">
    - Verifica que tu servidor tenga acceso a internet para alcanzar `api.fireworks.ai`
    - Comprueba si hay restricciones de firewall
    - Revisa los registros de la API/worker para mensajes de error detallados
    - Asegúrate de estar usando el endpoint de API correcto
  </Accordion>
  
  <Accordion title="Problemas de rendimiento">
    - Fireworks AI proporciona velocidades líderes en la industria con latencia mínima
    - Verifica tu conectividad de red para un rendimiento óptimo
    - Considera usar despliegues dedicados para cargas de trabajo empresariales
    - Monitorea tus patrones de uso para optimizar las llamadas a la API
  </Accordion>
  
  <Accordion title="Límite de tasa">
    - Fireworks AI proporciona altos límites de tasa en infraestructura sin servidor
    - Verifica tu uso actual en el panel de Fireworks AI
    - Considera actualizar a despliegues dedicados para mayor rendimiento
    - Contacta al soporte de Fireworks AI para ajustes de límite de tasa empresarial
  </Accordion>
</AccordionGroup>
