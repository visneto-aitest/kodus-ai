---

Cursor、GitHub Copilot、Claude、またはその他のAIコーディングツールにすでにルールが定義されている場合、それらをKodusにインポートして、コードレビュー中も同じ規約が適用されるようにできます。

## 動作方法

Kodusはリポジトリ内のルールファイルを検出し、Kodyルールとして同期することを提案します。サポートされているファイル形式には以下が含まれます：

- `.cursorrules`（Cursor）
- `.github/copilot-instructions.md`（GitHub Copilot）
- `.claude` ファイル（Claude）
- その他のAIツール設定ファイル

## セットアップ方法

1. **コードレビュー設定** → **Kodyルール** に移動します
2. Kodusがリポジトリ内の互換性のあるルールファイルを自動検出します
3. 検出されたルールを確認して、インポートしたいものを選択します
4. インポートされたルールはKodyルールとして追加され、コードレビュー中に適用されます

## なぜ重要か

同期しないと、IDEとコードレビューツールが異なる標準を適用します。Cursorのルールに従っている開発者が、IDEでは検出されなかったことでレビュー中にフラグされる可能性があります（またはその逆）。

インポートにより、ワークフロー全体でコーディング標準の**単一の信頼できる情報源**が確保されます。

詳細については、[IDEルールの同期](/how_to_use/ja/code_review/configs/rules_file_detection)を参照してください。
