GitHubにKodusをセットアップするには数分かかります。接続されると、すべてのプルリクエストが自動的にレビューされます。

## 手順

1. [kodus.io](https://kodus.io)で**アカウントを作成**するか、[Kodusをセルフホスト](/how_to_deploy/ja/deploy_kodus/generic_vm)します
2. **GitHubに接続** — セットアップ中にプロンプトが表示されたらKodus GitHub Appをインストールします
3. **リポジトリを選択** — Kodusがレビューすべきリポジトリを選択します
4. **PRをオープン** — Kodusが自動的にレビューしてコメントを投稿します

## 得られるもの

- すべてのPRでの自動レビュー（またはオンデマンドで `@kody start-review`）
- コード提案付きでPR内に直接投稿されるコメント
- カスタマイズ可能なルール、重要度フィルター、提案数制限
- リンクされたタスクに対するビジネスロジック検証

## カスタム設定

リポジトリごとの動作をカスタマイズするには、リポジトリルートに `kodus-config.yml` を配置します。ファイルはウェブ設定を自動的にオーバーライドします。

## 個人アクセストークンの使用

プライベートリポジトリやより厳格な権限を持つ組織では、GitHub個人アクセストークン（PAT）を設定する必要がある場合があります。

完全なセットアップガイドについては、[クイックスタート](/how_to_use/ja/quickstart)と[GitHub Appセットアップ](/how_to_deploy/ja/platforms/github/github_app)を参照してください。
