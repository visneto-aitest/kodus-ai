JiraをKodusに接続すると、2つの強力な機能が有効になります：Jiraチケットに対する自動**ビジネスロジック検証**と、コードレビュー提案のためのリッチなコンテキスト。

## 接続方法

1. **設定** → **プラグイン** に移動します
2. 利用可能なプラグインで**Jira**を見つけます
3. **接続**をクリックしてJiraワークスペースへのアクセスを認証します
4. Kodusがアクセスできるプロジェクトを選択します

## 有効になる機能

### 自動ビジネスロジック検証

PRがJiraチケットにリンクされると、Kodusは自動的に：
- チケットのタイトル、説明、受け入れ基準を取得
- PRのdiffを要件と比較
- 不足している実装、スコープの不一致、ギャップを報告

### オンデマンド検証

PRコメントに任意のJira URLを貼り付けます：

```
@kody -v business-logic https://your-org.atlassian.net/browse/PROJ-123
```

### リッチなコードレビューコンテキスト

Jiraが接続されると、Kodyはルール内のMCP関数を使用してタスクコンテキストを取得し、イシューのステータスを確認し、要件をクロスリファレンスできます。

## 要件

- Jira CloudまたはJira Data Center
- OAuthまたはAPIトークンアクセス
- 関連プロジェクトを読む権限

ビジネスロジック検証の詳細については、[ビジネスロジック検証](/how_to_use/ja/code_review/business_logic_validation)を参照してください。プラグインのセットアップについては、[プラグイン](/how_to_use/ja/code_review/plugins)を参照してください。
