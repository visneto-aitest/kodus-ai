## Cómo funciona Fireworks AI

Fireworks AI es la plataforma de inferencia más rápida para IA generativa, diseñada para construir y ejecutar aplicaciones de IA en segundos. La plataforma proporciona acceso serverless a modelos populares de código abierto como DeepSeek, Llama, Qwen y Mistral con velocidad optimizada, alto rendimiento y latencia mínima. Creada para desarrolladores que necesitan infraestructura de IA confiable y ultrarrápida sin la complejidad de gestionar GPUs.

## Crear clave de API

<Warning>Se requiere una cuenta de Fireworks AI para crear una clave de API.</Warning>

Ve directamente a la [Consola de Fireworks AI](https://app.fireworks.ai) para crear una nueva clave de API.

O sigue estos pasos:

1. Visita [app.fireworks.ai](https://app.fireworks.ai) y crea una cuenta o inicia sesión
2. Una vez dentro, navega a la página **API Keys** en la configuración de tu cuenta
3. Haz clic en el botón **"Create API Key"**
4. Dale a tu clave un nombre descriptivo (p. ej., 'Kodus' o el nombre que prefieras)
5. Haz clic en **"Create"** para generar la clave
6. Copia la clave de API inmediatamente y guárdala en un lugar seguro: no podrás verla de nuevo

<Info>
  Las cuentas nuevas vienen con $1 en créditos gratuitos para comenzar con tus proyectos.
</Info>

### Configura Fireworks AI en el archivo de entorno

Edita tu archivo `.env` y configura los ajustes principales. Para la **Integración con LLM**, usa Fireworks AI en Modo Fijo:

```env
# Core System Settings (update with your domains)
WEB_HOSTNAME_API="kodus-api.yourdomain.com"    
WEB_PORT_API=443                               
NEXTAUTH_URL="https://kodus-web.yourdomain.com"

# Security Keys (generate with openssl commands above)
WEB_NEXTAUTH_SECRET="your-generated-secret"
WEB_JWT_SECRET_KEY="your-generated-secret"
API_CRYPTO_KEY="your-generated-hex-key"
API_JWT_SECRET="your-generated-secret"
API_JWT_REFRESHSECRET="your-generated-secret"

# Database Configuration
API_PG_DB_PASSWORD="your-secure-db-password"
API_MG_DB_PASSWORD="your-secure-db-password"

# Fireworks AI Configuration (Fixed Mode) 
API_LLM_PROVIDER_MODEL="accounts/fireworks/models/llama4-maverick-instruct-basic"  # Choose your preferred model
API_OPENAI_FORCE_BASE_URL="https://api.fireworks.ai/inference/v1"                  # Fireworks AI API URL  
API_OPEN_AI_API_KEY="your-fireworks-api-key"                                       # Your Fireworks AI API Key

# Git Provider Webhooks (choose your provider)
API_GITHUB_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/github/webhook"
# or API_GITLAB_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/gitlab/webhook"
# or GLOBAL_BITBUCKET_CODE_MANAGEMENT_WEBHOOK="https://kodus-api.yourdomain.com/bitbucket/webhook"
```

<Note>
  Las URLs de webhook deben alcanzar el servicio de Webhooks (puerto 3332). Usa un dominio dedicado para webhooks o enruta `/.../webhook` al puerto 3332 en tu proxy inverso.
</Note>

<Info>
  **El Modo Fijo es ideal para Fireworks AI** porque proporciona APIs compatibles con OpenAI con velocidades de inferencia ultrarrápidas y acceso a modelos de código abierto de vanguardia con cero tiempo de configuración.
</Info>

### Configura el proxy inverso (para producción)

Para webhooks y acceso externo, configura Nginx:

```nginx
# Web App (port 3000)
server {
    listen 80;
    server_name kodus-web.yourdomain.com;
    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}

# API (port 3001)  
server {
    listen 80;
    server_name kodus-api.yourdomain.com;
    location ~ ^/(github|gitlab|bitbucket|azure-repos)/webhook {
        proxy_pass http://localhost:3332;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location / {
        proxy_pass http://localhost:3001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### Verifica la integración con Fireworks AI

Además de la verificación básica de instalación, confirma que Fireworks AI está funcionando:

```bash
# Verify Fireworks AI API connection specifically
docker-compose logs api worker | grep -i fireworks
```

  Para información detallada sobre la configuración de SSL, monitoreo y configuraciones avanzadas, consulta nuestra [guía de despliegue completa](https://docs.kodus.io/docs/how_to_deploy/en/deploy_kodus/generic_vm).

### Solución de problemas

<AccordionGroup>
  <Accordion title="Problemas con la clave de API">
    - Verifica que tu clave de API sea correcta y esté activa en la [Consola de Fireworks AI](https://app.fireworks.ai)
    - Comprueba si tienes créditos suficientes en tu cuenta de Fireworks AI
    - Asegúrate de que no haya espacios adicionales en tu archivo `.env`
    - Las cuentas nuevas reciben $1 en créditos gratuitos para comenzar
  </Accordion>
  
  <Accordion title="Modelo no encontrado">
    - Verifica que el nombre del modelo esté escrito correctamente en tu configuración
    - Confirma que el modelo esté disponible en la biblioteca de modelos actual de Fireworks AI
    - Prueba con un modelo diferente de nuestra lista recomendada
    - Consulta la [documentación de modelos de Fireworks AI](https://docs.fireworks.ai/getting-started/recommended-open-models)
  </Accordion>
  
  <Accordion title="Errores de conexión">
    - Verifica que tu servidor tenga acceso a internet para alcanzar `api.fireworks.ai`
    - Comprueba si hay restricciones de firewall
    - Revisa los logs de la API/worker para mensajes de error detallados
    - Asegúrate de estar usando el endpoint de API correcto
  </Accordion>
  
  <Accordion title="Problemas de rendimiento">
    - Fireworks AI proporciona velocidades líderes en la industria con latencia mínima
    - Comprueba tu conectividad de red para un rendimiento óptimo
    - Considera usar despliegues dedicados para cargas de trabajo empresariales
    - Monitorea tus patrones de uso para optimizar las llamadas a la API
  </Accordion>
  
  <Accordion title="Límite de velocidad">
    - Fireworks AI proporciona altos límites de velocidad en infraestructura serverless
    - Comprueba tu uso actual en el panel de Fireworks AI
    - Considera actualizar a despliegues dedicados para mayor rendimiento
    - Contacta al soporte de Fireworks AI para ajustes de límites de velocidad empresariales
  </Accordion>
</AccordionGroup>
